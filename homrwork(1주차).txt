1.버전관리하는방법
(1).git 과 source tree를 설치해준다
(2).바탕화면에 있는 폴더에 git init을 해준다
(3).sourcetree 에 가서 add페이지에서 추가해준다
(4).git 으로 이동해 , commit으로 변경,추가 등을 해준다
(5).history 에 잘 저장 되 있는지 확인한다


commit 이란?
source 들을 추가 변경 등을 해주는 장치이다
history 란?
commit 누른거를 저장해준다 , 무엇이 삭제 되고 , 무엇이 추가 되었으며,몇시몇분 누가변경했고
추가했고 까지 다 기록이남는다

2.협업관리 
(1).협업을 할려면 나만 보는것보다 다같이 봐야한다 그럴려면 다같이 볼 공간이 필요하다
(2).종류 :원격 repo
	local repo
원격 repo 란 ?
url 만 있으면 자료를 다볼수있는 상태 
local repo 란?
내 컴퓨터에서 만 볼수있는 자료 상태
